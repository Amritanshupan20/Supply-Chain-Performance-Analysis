# 📦 Supply Chain Performance Analysis

## 📌 Project Overview

This project analyzes supply chain performance using Python, PostgreSQL, SQL, and Power BI. It focuses on identifying key business insights related to sales, profit, customers, products, shipping, and delivery performance through data analysis and interactive dashboards.

---

## 🎯 Objectives

- Analyze sales and profit performance
- Identify top-performing product categories
- Evaluate shipping and delivery performance
- Perform SQL-based business analysis
- Build an interactive Power BI dashboard
- Generate actionable business insights

---

## 📈 Key Insights

- Fishing generated the highest sales among all product categories.
- Standard Class was the most frequently used shipping mode.
- A small number of products contributed a significant share of total revenue.
- Sales performance varied across different markets and regions.
- Delivery status analysis helped identify logistics performance trends.
